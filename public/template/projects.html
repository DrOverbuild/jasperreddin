MakeYourOwn.buzz

This is a BigCommerce template built for MakeYourOwn.buzz. BigCommerce built their own theme platform called Stencil, and it includes a complete CLI toolset for building, debugging, and bundling themes. This particular theme is a fork of BigCommerce's open-source default theme, Cornerstone. My specific changes include a redesigned navigation bar, header, and front page, as well an additional tab in the product section and other minor changes and improvements.

MakeYourOwn.buzz GitHub

ChemistryConnection.com

This site is a full stack web solution for storing and linking to technical data sheets on MakeYourOwn.buzz. Its purpose was to be an easy to use PDF hosting tool for the Chemistry Connection staff to browse, upload, and copy links to technical data sheets for products on MakeYourOwn.buzz. Built with the Apache PHP web server.

ChemistryConnection GitHub

missingthejoke.com

missingthejoke.com is small landing page for a podcast that I produce with a friend of mine. Includes links to listen to the podcast, view on Patreon, and a small webstore for buying our stickers. Hosted on Google Firebase. Built with Bootstrap and AngularJS. The store checkout uses the Stripe framework.

missingthejoke.com GitHub

cohoresource.app

Formerly cohoresourcebook.org, this website was built with a hand-built PHP template system, MySQL, HTML, CSS, JavaScript (with JQuery). It was built for administrators and interns at the City of Hope Outreach (a local non-profit in Conway, AR) to easily add and manage content of the resource app built for iOS and Android. Hosting and data has been migrated to Google Firebase, and there are plans in the works to rebuild the site as a single-page web app using AngularJS.
